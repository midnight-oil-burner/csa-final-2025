Unit 4: Iteration

4.1: while Loops

while loops execute the code inside them as long as the boolean expression is true: 

    while(booleanExpression)
    {
        // code runs while booleanExpression is true
    }

In order to prevent infinite loops, an outside variable called a loop variable needs to be created and iterated. The boolean evaluating as false or a return statement being executed can exit out of a while loop. While loops are generally used if you're not sure how many times it will need to run.

4.2: for Loops

for loops are like while loops, but you intialize and iterate the loop variable within the loop. They're generally used when you know how many times it'll need to run. 

    for(initialization, boolean condition, iteration)
    {
        // initialization initializes a loop variable, which then iterates as iteration defines and runs the code block while the boolean condition is true
        // iteration must be a ++ or -- or += or -= etc.
    }
    for(int i = 0; i < 2; i++) // starts i at 0, adds 1 to i every loop, runs code while i < 2
    {
        System.out.println(i); // prints numbers 0 & 1
    }
    // make note that the above stops before i reaches 2. if the boolean was i <= 2, it would run thrice.

4.3: Developing Algorithms with Strings 

    // To loop through an entire string:
    for(int i = 0; i < str.length(); i++)
    {
        System.out.println(str.substring(i, i+1));
    }
    // the above stops right before str.length(), so an out of bounds exception doesn't occur
    // combined with String methods a lot of algorithms can be made, for example:
    for(int i = str.length()-1; i >= 0; i--)
    {
        System.out.println(str.substring(i, i+1));
    }
    // prints a String in reverse
    // there will definitely be a lot of these on the exam

4.4: Nested Iteration

Loops can be placed inside other loops. Inner loops are executed before outer loops, and the total number of times a nested loop will run can be found by multiplying the number of times the outer loop will run by the number of times the inner loop will run. 

    for(int i = 0; i < 5; i++)
    {
        for(int j = 0; i < 5; j++)
        {
            System.out.println(i);
            System.out.println(j);
        }
    }
    // outer runs 5 times, inner runs 5 times, total is 25 loops
