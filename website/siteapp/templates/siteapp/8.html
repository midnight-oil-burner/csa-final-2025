Unit 8: 2D Array

8.1: 2D Arrays

2D arrays are arrays of arrays. For this exam, all of the arrays in the array will be the same length. 

    // initialization
    int[][] nums = {array1, array2, array3};        // uses array names
    int[][] digits = {{1,2,3},{4,5,6},{7,8,9}};     // uses newly made arrays
    int[][] integers = new int[2][3];               // creates an empty 2D array with 2 rows and 3 columns 
    // the first number is the row number (horizontal), the second is the column number (vertical)

    // to modify/access elements
    int x = digits[0][1];       // accesses 1st array's 2nd element
    digits[1][2] = 12;          // changes the 2nd array's 3rd element to 12

When traversing through 2D arrays, there are two different ways: row-major order, which loops through the horizontal rows/individual arrays, and column-major order, which loops through the vertical columns. This exam generally uses row-major order.

8.2: Traversing 2D Arrays

2D arrays can be traversed with nested loops: 

    // row-major order
    for(int row = 0; row < arr.length; row++) 
    {
        for(int col = 0; col < arr[0].length; col++) 
        {
            System.out.println(arr[row][col]);
        }
    }
    
    // column-major order
    for(int col = 0; col < arr.[0].length; col++)
    {
        for(int row = 0; row < arr.length; row++)
        {
            System.out.println(arr[row][col]);
        }
    }

    // enhanced for loops can also be used for access:
    for(int[] singleArray : arr)
    {
        for(int singleElement : singleArray)
        {
            System.out.println(singleElement);
        }
    }

Array algorithms can also be rewritten for 2D arrays. Sequential searching algorithms have to traverse the array as shown above.
