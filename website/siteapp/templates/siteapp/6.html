Unit 6: Array

6.1: Array Creation and Access

An array is an object that can store multiple primitve values and objects of the same type. They use square brackets: 

    // array declaration
    int[] scores = new int[5]; // creates a new int array of size 5, with each value being set to their default value
    int[] nums = {6,7,8}; // creates a new int array of size 3, with 6, 7, and 8 being stored
    // you can also initialize new objects inside arrays:
    Type[] nums = {new Type(param1, param2), new Type(param3, param4)};

    // array access
    // arrays use indices like Strings do; 0 refers to the first index, array.length-1 refers to the last
    // accessing index array.length throws an error
    array[index] = value
    scores[0] = 6;

    // array.length returns the length of the array; it's a public variable and not a method, hence the lack of parentheses
    // the size of an array cannot be changed without another declaration

    // parallel arrays are arrays that store related information across multiple arrays
    String[] firstNames = {"John", "Greg"};
    String[] lastNames = {"Greene", "Heffley"};

    // arrays are objects, so passing them as an argument into a method can change the original array

6.2: Traversing Arrays

Arrays can be traversed with loops, which allows easy accessing and altering: 

    int[] arr = {1,2,3,4};
    for(int i = 0; i < arr.length; i++) // traversing with a for loop
    {
        System.out.println(arr[i]);
    }

    int track = 0;
    while(track < arr.length) // traversing with a while loop
    {
        System.out.println(arr[track]);
        track++;
    }

6.3: Enhanced for Loop for Arrays

You can use an enhanced for loop to access all elements of an array easily. However, you cannot alter arrays or access indices with them. 

    for(type x: array)
    {
        ...
    }

    for(int x:nums) // prints all ints in array nums
    {
        System.out.println(x);
    }
